About Error Garden

Error Garden is an experimental music album that transforms software system errors into musical compositions. Using Data Sonification and Algorithmic Composition, each track represents a different type of programming error, turning technical "failures" into aesthetic experiences.

Techniques: Data Sonification, Algorithmic Composition, Physical Modeling, Generative Grammars

Technology: Web Audio API, Python, SuperCollider, FAUST, Csound
